Who I am

Here is the introduction...

This is a simple blog website I build with all open source projects, to experience the Java architecture design for Web and mobile applications. The whole system is built upon Spring Framework, Spring Security, Spring Social, Spring Data, and some other open source projects for front end Web UI, like Bootstrap, AngularJS etc. All data is stored in a NoSQL database - MongoDB. The Web application is deployed to Cloud Foundry (Running at Pivotal Web Services), an open source PAAS cloud platform.
